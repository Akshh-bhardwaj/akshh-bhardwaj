## ❓ Interview Questions
<details>
  <summary><b>1. What is the difference between malloc() and calloc() in C?</b></summary>
  <br>
  <code>malloc()</code> allocates uninitialized memory block, while <code>calloc()</code> allocates memory and initializes all bits to zero. <code>calloc()</code> takes two arguments (num of items, size of each), whereas <code>malloc()</code> takes one (total bytes).
</details>

<details>
  <summary><b>2. How does the Event Loop work in JavaScript?</b></summary>
  <br>
  The Event Loop constantly checks if the call stack is empty. If it is, it looks at the callback queue (and microtask queue) and pushes the first task onto the stack for execution. This is how JS handles asynchronous operations in a single-threaded environment.
</details>

<details>
  <summary><b>3. Explain the Virtual DOM in React.</b></summary>
  <br>
  The Virtual DOM is a lightweight copy of the actual DOM. React uses it to diff changes (reconciliation) and update only the specific parts of the actual DOM that have changed, rather than re-rendering the entire page, significantly improving performance.
</details>
